## Principios de Diseño Material Design 3

### 1. Elevación y Sombras
- Usa los tokens de superficie (`surface-container-*`) para crear jerarquía visual
- Las sombras deben ser sutiles y usar el color `shadow` definido
- La elevación se logra mediante cambios de color de superficie, no solo sombras

### 2. Estados Interactivos
- **Hover**: Aplica overlay con opacidad del 8-12%
- **Focus**: Usa `outline` con grosor de 2-3px
- **Pressed**: Aplica overlay con opacidad del 12-16%
- **Disabled**: Reduce la opacidad al 38% del color original

### 3. Tipografía
- **Display Large**: 57px / 64px line height
- **Display Medium**: 45px / 52px line height
- **Display Small**: 36px / 44px line height
- **Headline Large**: 32px / 40px line height
- **Headline Medium**: 28px / 36px line height
- **Headline Small**: 24px / 32px line height
- **Title Large**: 22px / 28px line height
- **Title Medium**: 16px / 24px line height
- **Title Small**: 14px / 20px line height
- **Body Large**: 16px / 24px line height
- **Body Medium**: 14px / 20px line height
- **Body Small**: 12px / 16px line height
- **Label Large**: 14px / 20px line height
- **Label Medium**: 12px / 16px line height
- **Label Small**: 11px / 16px line height

### 4. Espaciado
- **Baseline grid**: 4px
- **Spacing scale**: 4px, 8px, 12px, 16px, 24px, 32px, 40px, 48px, 64px, 80px
- **Padding interno de componentes**: Generalmente múltiplos de 8px

### 5. Bordes Redondeados
- **None**: 0px
- **Extra Small**: 4px
- **Small**: 8px
- **Medium**: 12px
- **Large**: 16px
- **Extra Large**: 28px
- **Full**: 9999px (círculo completo)

### 6. Componentes Clave

#### Botones
- **Filled**: Usa `primary` con `on-primary`
- **Outlined**: Usa `outline` con `primary` para texto
- **Text**: Usa `primary` sin fondo
- **Elevated**: Usa `surface-container-low` con sombra sutil
- **Tonal**: Usa `secondary-container` con `on-secondary-container`

#### Cards
- Usa `surface-container-highest` o `surface-container` para el fondo
- Añade borde redondeado medium (12px)
- Padding interno de 16-24px

#### Campos de Texto
- Usa `surface-container-highest` para el fondo
- Borde con `outline` en estado normal
- Borde con `primary` en estado focus
- Label con `on-surface-variant`

#### FAB (Floating Action Button)
- Usa `primary-container` con `on-primary-container`
- Elevación pronunciada
- Borde redondeado large (16px)

## Uso de Colores según Contexto

### Énfasis Alto
- Usa colores `on-*` principales (e.g., `on-surface`, `on-primary`)
- Opacidad: 100%

### Énfasis Medio
- Usa colores `on-surface-variant` o `on-*-container`
- Opacidad: 74%

### Énfasis Bajo
- Usa colores `outline`
- Opacidad: 38%

### Estados Deshabilitados
- Opacidad: 38% del color original
- Usa `on-surface` con opacidad reducida

## Accesibilidad

### Contraste de Color
- **Texto normal**: Mínimo 4.5:1
- **Texto grande**: Mínimo 3:1
- **Elementos UI**: Mínimo 3:1

### Áreas Táctiles
- **Mínimo**: 48x48px para elementos interactivos
- **Recomendado**: 56x56px para acciones primarias

### Estados de Foco
- Siempre visible y con alto contraste
- Usar `outline` del tema

## Integración con IA Chatbot

### Recomendaciones de Diseño
1. **Burbujas de chat**: Usar `primary-container` para mensajes del usuario y `surface-container-high` para respuestas del bot
2. **Indicador de escritura**: Usar animación con `primary` color
3. **Input de texto**: Usar `surface-container-highest` con borde `outline`
4. **Avatares**: Circular (border-radius: full) con `primary` o `secondary` como fondo
5. **Timestamps**: Usar `on-surface-variant` con énfasis bajo
6. **Estados de carga**: Usar skeleton screens con `surface-variant`
